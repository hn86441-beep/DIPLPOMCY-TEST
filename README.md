# 🕊️ بوابة الإعداد لامتحان السلك الدبلوماسي
تطبيق تفاعلي مبني بـ Streamlit للاستعداد لامتحان السلك الدبلوماسي، مع تركيز خاص على **اللغة الروسية** و**الترجمة الدبلوماسية**.

## ✨ الميزات
- 📚 **قاموس مصطلحات** ثلاثي اللغة (عربي/إنجليزي/روسي) مع بحث وتصفية حسب الفئة.
- 🎴 **بطاقات تعلّم تفاعلية (Flashcards)** بنظام "أتقنتها / أحتاج مراجعة"، ورفع ملف CSV خاص بك.
- 🧠 **اختبار ذاتي سريع** (مصطلح ↔ ترجمة روسية) بأسئلة عشوائية.
- 🌐 **محاكي ترجمة دبلوماسية** بنصوص حقيقية الطابع (بيانات، مذكرات تفاهم، تصريحات وزارية).
- 🤖 **محرك تقييم ترجمة ذكي** يحلل: التشابه الدلالي، تغطية المصطلحات، الاكتمال، والسجل الرسمي — بتقرير ثلاثي اللغة، مع خيار تقييم متقدم عبر Claude API (اختياري، بمفتاحك الخاص).
- 📝 **اختبارات تحضيرية**: أسئلة اختيار من متعدد وأسئلة مقالية بنمط العلاقات الدولية.
- 🏆 **لوحة تقدم (Dashboard)**: تتبع تقدمك، سلسلة المذاكرة، متوسط درجات الترجمة عبر الزمن، ونصيحة يومية.

## 🗂️ هيكلة المشروع

```
├── app.py                     # التطبيق الرئيسي
├── requirements.txt           # المتطلبات

│   ├── __init__.py

```

## 🚀 التشغيل محلياً
```bash
git clone https://github.com/USERNAME/diplomatic-prep-app.git
cd diplomatic-prep-app
python -m venv venv
source venv/bin/activate      # على ويندوز: venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
سيفتح التطبيق تلقائياً على `http://localhost:8501`.

## ☁️ دليل النشر على GitHub + Streamlit Community Cloud

### 1) رفع المشروع على GitHub
```bash
cd diplomatic_prep_app
git init
git add .
git commit -m "إطلاق النسخة الأولى من بوابة الإعداد للسلك الدبلوماسي"
git branch -M main
git remote add origin https://github.com/USERNAME/diplomatic-prep-app.git
git push -u origin main
```
> استبدل `USERNAME` باسم المستخدم الخاص بك على GitHub، وتأكد من إنشاء مستودع (Repository) فارغ بنفس الاسم أولاً من واجهة GitHub.

### 2) النشر على Streamlit Community Cloud (مجاناً)
1. اذهب إلى **https://share.streamlit.io** وسجّل الدخول بحساب GitHub.
2. اضغط **"New app"**.
3. اختر المستودع (Repository) الذي رفعته، والفرع `main`.
4. في خانة **"Main file path"** اكتب: `app.py`
5. اضغط **"Deploy"** وانتظر بضع دقائق حتى تُثبَّت المتطلبات ويعمل التطبيق.
6. ستحصل على رابط دائم بصيغة:
   `https://USERNAME-diplomatic-prep-app.streamlit.app`

### 3) (اختياري) تفعيل ميزة التقييم المتقدم بالذكاء الاصطناعي
إن أردت تفعيل التحليل العميق عبر Claude API لكل المستخدمين دون إدخال مفتاح يدوياً، أضف مفتاحك كـ **Secret** آمن:
1. من لوحة تحكم التطبيق على Streamlit Cloud → **Settings → Secrets**.
2. أضف:
   ```toml
   ANTHROPIC_API_KEY = "sk-ant-xxxxxxxx"
   ```
3. عدّل في `app.py` سطر قراءة المفتاح إلى:
   ```python
   import os
   default_key = st.secrets.get("ANTHROPIC_API_KEY", "")
   ```
   (تُبقي حقل الإدخال اليدوي كخيار بديل للمستخدمين الآخرين).

## 🧩 تخصيص المحتوى
- لإضافة مصطلحات جديدة: عدّل `data/glossary.csv` مباشرة (نفس الأعمدة).
- لإضافة نصوص ترجمة جديدة: عدّل `data/diplomatic_texts.csv` — احرص على كتابة `model_translation` احترافي لأن محرك التقييم يقارن به.
- لإضافة أسئلة اختبار: عدّل `data/mcq.csv`.

## ⚙️ كيف يعمل محرك تقييم الترجمة؟ (`utils/evaluator.py`)
يجمع 4 مؤشرات مرجّحة لإنتاج درجة من 100:
| المؤشر | الوزن | الوصف |
|---|---|---|
| التشابه الدلالي | 40% | تشابه TF-IDF/Cosine بين ترجمتك والنموذج |
| تغطية المصطلحات | 30% | هل استخدمت المصطلحات الدبلوماسية الصحيحة من القاموس؟ |
| الاكتمال | 15% | هل ترجمتك مكتملة دون حذف أو إطناب مفرط؟ |
| السجل الرسمي | 15% | هل حافظت على الأسلوب الدبلوماسي الرسمي (وتجنبت العامية)؟ |

النتيجة تُعرض بتقرير مفصّل بثلاث لغات + عرض فروقات نصية (Diff) + قائمة بالمصطلحات الناقصة، مع خيار تحليل إضافي عبر الذكاء الاصطناعي إن توفر مفتاح API.
